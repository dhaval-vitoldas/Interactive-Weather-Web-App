# Interactive Weather Web App
Full-stack weather app built using Python, Flask and Leaflet.js

## How to run
1. Install required libraries:
    pip install flask requests
2. Go to https://openweathermap.org/, create a free account and get your API key.
3. Paste your API key in weather.py 
4. Run the code:
    python weather.py
5. Go to http://127.0.0.1:5000/ to use the app!
